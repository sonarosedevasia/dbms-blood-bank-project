# Blood Bank Management System

A Flask-based DBMS mini project for managing blood donor records.

## Features
- Add donor details
- Update donor information
- Delete donor records
- Search donors by ID, name, and blood group

## Technologies Used
- Python
- Flask
- SQLite
- HTML, CSS

## How to Run
1. Install Python
2. Install Flask
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000
